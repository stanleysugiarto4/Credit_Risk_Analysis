# Credit Risk Portfolio Analysis Using SQL

This project analyzes a consumer lending portfolio using PostgreSQL to identify borrower and loan characteristics associated with higher default risk. The analysis focuses on portfolio-level KPIs, borrower segmentation, default-rate comparison across demographic and credit-risk groups, and underwriting policy simulation.

The goal of the project is to answer three key business questions:

1. Which borrower and loan segments are most likely to default?
2. Where is default exposure concentrated in the portfolio?
3. How can underwriting rules help reduce risky loan approvals while preserving business volume?

The project includes data cleaning, feature engineering, demographic risk analysis, credit-risk segmentation, and simulated underwriting decisions such as Approve, Manual Review, and Reject.
